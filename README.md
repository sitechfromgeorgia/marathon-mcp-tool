## ­ЪїЪ рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў

- **РџА 80+ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљ 7 рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃљрЃерЃў** - рЃАрЃарЃБрЃџрЃў рЃћрЃЎрЃЮрЃАрЃўрЃАрЃбрЃћрЃЏрЃљ AI рЃљрЃАрЃўрЃАрЃбрЃћрЃюрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА

## ­Ъїі рЃЉрЃљрЃЌрЃБрЃЏрЃБрЃарЃў рЃ«рЃћрЃџрЃгрЃћрЃарЃљ

Marathon MCP Tool рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ **рЃАрЃўрЃДрЃЋрЃљрЃарЃБрЃџрЃўрЃЌ рЃЉрЃљрЃЌрЃБрЃЏрЃерЃў, рЃАрЃљрЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃЮрЃерЃў** ­ЪЄг­ЪЄф

### рЃЕрЃЋрЃћрЃюрЃў рЃдрЃўрЃарЃћрЃЉрЃБрЃџрЃћрЃЉрЃћрЃЉрЃў:

## ­ЪцЮ рЃгрЃЋрЃџрЃўрЃџрЃў

### рЃЎрЃЮрЃюрЃбрЃарЃўрЃЉрЃБрЃфрЃўрЃўрЃА рЃгрЃћрЃАрЃћрЃЉрЃў:

- **РюЁ рЃбрЃћрЃАрЃбрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃЋрЃћрЃарЃљрЃЪрЃў** - рЃЏрЃўрЃюрЃўрЃЏрЃБрЃЏ 80%
