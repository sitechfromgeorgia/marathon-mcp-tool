## 🔄 Migration Guide v1.x → v2.0.0

### ძირითადი ცვლილებები:
```typescript
// v1.x
import { MarathonTool } from 'marathon-mcp-tool';

// v2.0.0
import { MarathonMCPServer } from 'marathon-mcp-tool';
const server = new MarathonMCPServer();
```

### ახალი Installation:
```bash
git clone https://github.com/sitechfromgeorgia/marathon-mcp-tool.git
cd marathon-mcp-tool
npm install
npm run build
npm start
```
